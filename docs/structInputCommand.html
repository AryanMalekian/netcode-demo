Detailed Description

Represents a timestamped input command for prediction systems.

Each input command captures user input state at a specific moment, tagged with sequence number and timing information. This enables the prediction system to replay exact input sequences during server reconciliation for deterministic state reconstruction.

Features:

  • Sequence-based ordering and acknowledgment
  • Normalized velocity inputs for consistent behavior
  • Frame timing for accurate replay simulation
