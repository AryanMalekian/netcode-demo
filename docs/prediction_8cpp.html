Detailed Description

Implementation of advanced client-side prediction for the netcode demo.

Implements both the legacy predictPosition() function (simple linear extrapolation) and the PredictionSystem class for robust, modern client-side prediction with input buffering, server reconciliation, and smooth error correction.
